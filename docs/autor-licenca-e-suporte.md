## Publicação no GitHub

Ao publicar como repositório público, recomenda-se incluir na raiz:

- `README.md`
- `LICENSE`
- `AUTHORS.md`
- `.env.example` (sem segredos)
- `backup.config.example.json`

Nunca commitar `.env`, `backups/` nem credenciais reais.
